# Processed canonical table outputs

`signal-validation build-wr-tables` writes deterministic canonical WR CSV outputs to this directory by default:

- `wr_player_weeks.csv`
- `wr_player_seasons.csv`
- `wr_feature_seasons.csv`
- `wr_outcome_seasons.csv`

These files are generated artifacts and depend on the raw historical CSV provided at ingestion time.
